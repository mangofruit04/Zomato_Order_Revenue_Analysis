# Zomato Order & Revenue Analysis

## Project Overview
Analysis of 150 restaurant orders across Mumbai, Pune and Delhi
using Excel and an interactive Power BI Dashboard.

##  Tools Used
- Microsoft Excel
- Power BI Desktop (KPI Cards, Pie, Column Charts, Slicer)

##  Key Visuals
- KPI Cards — Total Orders, Revenue, Avg Rating
- Pie Chart — Online vs Offline Orders
- Column Chart — Revenue by Cuisine
- City Slicer — Dynamic filtering across all visuals

##  Key Insights
- Identified top performing city by order volume
- Found highest revenue generating cuisine category
- 65% orders are Online — digital channel dominates
- Average customer rating of 4.0/5.0 across all cities
